Description:	Roundtrip application to test dwg and dxf operations between Icad and Acad.
Comments:	This roundtrip application allows users to test different versions of Icad and Acad
		for roundtrip compatibility. This allows an user to select Icad or Acad for any step,
		choose different executables for each step (Icad 2001, ACAD R14, etc.), decide on the
		either dxf or dwg as output format. In addition the version to either do a dxfout or
		saveas can be chosen. The output of one step in the input to the next step and a 
		maximum of five steps can be chosen.
