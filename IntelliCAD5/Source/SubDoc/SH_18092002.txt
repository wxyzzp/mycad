Description:	This submission integrate the imagemenu code with IntelliCAD.

Comments:	The imagemenu was implemented as a SDS add-on application, which needed
                to be manually loaded into IntelliCAD by using (xload "imagemenu") every 
                time when a new section is started. This submission moves this functionality
                into IntelliCAD. A flag BUILD_WITH_IMAGEMENU_SUPPORT is added to 
                configure.h so a member who doesn't want to built it, has the choice to 
                not build it. This submission also fixed bugs in origianl code.

Usage:          See imagemenu.txt.
